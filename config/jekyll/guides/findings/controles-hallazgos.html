Controles al crear un hallazgo

Los controles descritos a continuación se ejecutan al momento de guardar el hallazgo:

  • Si tiene el estado Implementado/Auditado, debe tener Fecha de solución y Respuesta
  • Si el estado Implementado, debe tener Fecha de implementación, Respuesta y no debe tener Fecha de solución
  • Si tiene el estado En Proceso de implementación, debe tener Fecha de implementación, Respuesta y no debe tener Fecha de solución
  • Si tiene el estado Riesgo asumido, debe tener Fecha de solución
