Crear un usuario

Para dar de alta un usuario debe completar los siguientes campos:

1

Usuario: por defecto el usuario se define con la primer letra del nombre y el apellido. Puede modificarse a pedido del usuario

2

Nombre del usuario

3

Apellido del usuario

4

Email: correo de la organización a la que pertenece

5

Cargo: puesto que ocupa en la organización

6

Notas: detalle de las modificaciones que se realizan en la parametrización del usuario

7

Tipo de recurso: función que cumple el usuario, se utiliza para realizar lso cálculos de costos de los recursos

8

Lenguaje: se utiliza por defecto Español

9

Conectado: se utiliza para ver si está conectado el usuario

10

Habilitado: este campo se habilita cuando se da de alta al usuario. Pasa a deshabilitado cuando caduca la contraseña del usuario. Automáticamente se habilita cuando el usuario restablece su contraseña.

11

Oculto: se utiliza en la baja del usuario, para que el mismo se encuentre oculto en los campos de búsqueda de usuarios

12

Enviar notificación: cuando se tilda esta opción al usuario le llega un correo notificándolo de que fue dado de alta y puede acceder a través del link indicado en el correo

13

Superior: se coloca el superior del usuario a los fines de definir la estructura jerárquica de la organización

14

Personal a cargo: se colocan usuarios que estén a cargo del usuario en cuestión. El usuario puede ver los Hallazgos de todas las personas que tenga a cargo

15

Usuario relacionado: se coloca un usuario relacionado a los fines que el usuario en cuestión pueda ver los hallazgos en los que se encuentra asignado el usuario relacionado

16

Organizaciones: se agregan las organizaciones en las que el usuario tenga actividad y el perfil correspondiente a la misma
